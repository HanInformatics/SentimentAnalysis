# SentimentAnalysis

**[ Members: 조은경, 김요욱, 구유진 ]**

We study sentiment analysis as "social pathology through languages". 
So, we scrape psychiatric news and terminology and analyse the data so as to mine sentiment information with psychiatric texts.
